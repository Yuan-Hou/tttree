【画风圣经 / Visual Style Bible】

整体风格:
氛围写实的数字概念插画。介于写实与绘画之间——有照片级的光影与材质质感，
但保留可见的笔触感和艺术化的色彩处理，而非纯照相写实。

光影:
强调戏剧性的自然光与环境光。善用明暗对比营造氛围，光源方向明确，
阴影柔和而有层次。倾向于一天中光线富有表情的时刻（清晨、黄昏、夜晚的人造光）。

色彩:
低饱和、偏电影感的调色。以一到两种主导色调统一整幅画面的情绪，
避免杂乱鲜艳的撞色。冷暖对比用于引导视觉焦点。

构图:
电影镜头式构图，有明确的前景、中景、远景层次，营造空间纵深。
视角通常为第二人称代入感的视高（如同主角正站在此处环顾）。

质感:
注重材质的真实触感——石头的粗粝、木头的纹理、金属的反光、布料的褶皱、
水汽与尘埃在光中的颗粒感。

角色呈现:
角色依据其 appearance 描述及参考图库中的立绘呈现，可正面、可特写。
重点是借助参考图保持角色形象在多张图之间的一致性。

排除项:
不要文字、不要画框边框、不要 UI 元素、不要拼贴、不要现代品牌物件
（除非情节明确要求）、不要卡通夸张比例、不要过曝的鲜艳色。
